# 遇到的问题

### el-upload 组件的排版问题

`el-upload`组件中的按钮无法像单独按钮一样排版，故选择隐藏内部按钮，使用外部按钮调用其 click 方法。

### 动态改变表格内容显示方式

想通过 checkbox 对账单分类、账单类型进行转换（展示中文），但尚未找到合理方法直接改变已经被 prop 绑定的数据
